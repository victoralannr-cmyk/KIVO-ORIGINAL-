# **App Name**: AetherAI

## Core Features:

- AI Agent Simulation: Simulate a generative AI agent to demonstrate its capabilities in a chat interface.
- Contact Form Submission: Submit contact form data to Firebase Firestore for lead management.
- Success Case Display: Showcase success stories with client testimonials and animated logos in an auto-sliding format.
- Benefits Showcase: Display key benefits using animated icons and concise descriptions.
- Animated Hero Section: Create a captivating hero section with an animated gradient background, floating particles, and simulated chat messages featuring parallax effects.
- AI Agent Integration Tool: Informative text about integrating AI into a company. Use AI as a tool to customize text depending on use-case and current events
- Homepage: All content is contained on a single homepage with internal navigation

## Style Guidelines:

- Primary color: Neon blue (#39C0ED) to convey innovation and trust.
- Background color: Dark, desaturated blue (#0A192F) to create a futuristic and professional feel.
- Accent color: Cyan (#64FFDA) to highlight interactive elements and calls to action.
- Font pairing: 'Space Grotesk' (sans-serif) for headings and 'Inter' (sans-serif) for body text. These modern sans-serif fonts will provide a clean and tech-forward look.
- Note: currently only Google Fonts are supported.
- Use smooth, animated icons to represent benefits and features, enhancing the interactive feel.
- Implement a modern layout inspired by Kyon Digital, featuring smooth animations, parallax effects, and glassmorphism.
- Apply 'scroll reveal' effects to all sections, creating a dynamic and engaging user experience.
- Use subtle parallax effects on images to add depth and visual interest.
- Incorporate hover microinteractions on buttons and texts for enhanced interactivity.
